2.1 Keyboard Accessible

October 17th, 2016

Definition

“Make all functionality available from a keyboard” - World Wide Web Consortium.

What's that mean?

Every function throughout your website should have the ability to be accessed and/or completed via a keyboard. A user should be able to tab through all content and functions of your website.

Why?

If all functionality of your website can be supported by a keyboard, it allows for flexibility of interaction. Keyboard support allows for the use of assistive technologies that simulate keystrokes as their output, such as speech to text, as well as by mouse using on-screen keyboards. This level of flexibility is beneficial to a wide range of users including those with a disability and may not be able to interact with your website as you envision.

Example

An example of keyboard accessibility would be in cases where elements only show up on hover. For instance, hovering over a link might show a start to allow the user to “Favorite” that link. It’s important that when a user tabs through the content using a keyboard, that they are also able to select the link to “Favorite” it. Some users might not use a mouse and would have no way of hovering over the link.

Keyboard Trap

Though technically a guideline underneath section 2.1, preventing users from a keyboard trap is extremely important. To explain the importance of this subsection, I will turn to Facebook. Everyone is pretty familiar with Facebook and their interface at this point making this a very relatable example. Facebook has 3 different columns of content on the desktop layout. If you were to tab through, expecting to tab from the column on the far left, and move to the far right, you will be stuck in a trap of tabbing through posts. Because Facebook continuously loads posts, you would be tabbing forever and would never reach the far right column content. They have prevented this by moving from the column on the far left, to the column on the far right, and then to the center column with all of the posts. This is important to be aware of to ensure all content and functionality can be accessed.
